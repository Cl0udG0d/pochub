# pochub

## 简介
`pochub` 用来记录互联网上已公开的漏洞检测poc，提供一个快速查找漏洞对应检测poc的平台

漏洞检测poc在`/app/database/pochub.db`文件中，为了方便部署使用`sqlite`数据库
## 演示demo

## 自行搭建

```
git clone https://github.com/Cl0udG0d/pochub
cd pochub
docker-compose up -d 
```
